# databricks-demand-forecast-pipeline
# Food Bank Demand Forecasting Pipeline

This project builds an end-to-end data pipeline and demand forecasting system using Databricks, Spark, and Delta Lake. The pipeline processes raw operational data through a Bronze–Silver–Gold architecture, performs data transformations and feature engineering, and trains a machine learning model to predict future demand.
The project also includes a Power BI dashboard to visualize demand trends, model performance, and future forecasts, enabling data-driven operational planning.


## Tech Stack

- Databricks
- Apache Spark
- Delta Lake
- PySpark
- Machine Learning (Linear Regression)
- Power BI

## Architecture

Bronze Layer  
Raw data ingestion and storage.

Silver Layer  
Data cleaning, validation, and transformation.

Gold Layer  
Aggregated analytical tables used for modeling and dashboards.

Machine Learning  
A regression model is trained to forecast weekly demand by location.

Visualization  
Power BI dashboard built for demand analysis and forecasting insights.
